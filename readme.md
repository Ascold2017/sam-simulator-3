# AA-Simulator 3.0

Pseudo-simulator of the surface-to-air system, based on Vue 3, TresJS, Cannon-es, Express

## Used technologies
- Vue 3
- TresJs (Three.js wrap)
- Cannon-ES
- Konva.js (VueKonva)
- Typescript
- Vuetify
- Pinia
